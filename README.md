# Student-Performance-Analytics-Dashboard
Synthetic student data for MSc Data Science project - Building a Student Performance Analytics Dashboard

## What This Is
- 9 years of synthetic student data (2017-2025)
- 3,038 students across 7 programmes
- Generated for my MSc thesis dashboard

## Files
- Year folders: Individual academic years
- COMBINED files: All years merged
- Scripts: Data generator and verification
